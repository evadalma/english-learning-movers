# English Learning Movers

Landing page for English Learning Movers, a language academy in Cartagena. This project demonstrates a responsive design with functional dropdown menus, call-to-action buttons, and accessible content for an optimal user experience.

## Features

- Multi-level navigation menu with dropdowns for different levels (Babies, Children, Adolescents)
- Responsive layout for mobile, tablet, and desktop
- Call-to-action buttons strategically placed to improve conversion
- Accessible images with alt attributes
- Semantic HTML structure for better readability and SEO
- Sidebar content with highlights, quotes, and key information

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts (Nunito)
- Material Icons
